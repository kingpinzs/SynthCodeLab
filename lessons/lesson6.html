Lesson 3: Displaying a Character on Screen

Step 1: Update Bootloader to Display a Character Let's enhance your bootloader to display a single character on the screen. We'll use BIOS interrupts to handle this.

Example Code:

[org 0x7c00] ; Set the origin to 0x7c00, which is where the BIOS loads the boot sector

; Clear the screen
mov ah, 0x00 ; mov: move data, ah: upper 8 bits of the AX register, 0x00: video mode 0 (clear screen)
mov al, 0x03 ; al: lower 8 bits of the AX register, 0x03: clear screen and move cursor to upper left
int 0x10     ; int: interrupt, 0x10: BIOS video service

; Print a single character 'A'
mov ah, 0x0E ; mov: move data, ah: upper 8 bits of the AX register, 0x0E: teletype output
mov al, 'A'  ; al: lower 8 bits of the AX register, 'A': ASCII code for character 'A'
int 0x10     ; int: interrupt, 0x10: BIOS video service

; Infinite loop to keep the OS running
hang:
    jmp hang ; jmp: jump to a label, hang: the label to jump to

times 510-($-$$) db 0  ; Fill the rest of the boot sector with zeros
dw 0xAA55              ; Boot sector signature
