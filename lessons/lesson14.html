Lesson 11: Writing a Kernel in C

Step 1: Create a Simple Kernel in C Let's write a simple kernel in C that prints a message to the screen. Create a file called kernel.c in the kernel directory:

Example Code:

void kmain(void) {
    const char *str = "Hello from Kernel!";
    char *vidptr = (char*)0xb8000;  // Video memory starts here
    unsigned int i = 0;
    unsigned int j = 0;

    // Clear the screen by writing spaces to video memory
    while(j < 80 * 25 * 2) {
        vidptr[j] = ' ';
        vidptr[j+1] = 0x07;  // Attribute byte - light grey on black background
        j = j + 2;
    }

    j = 0;

    // Write the string to video memory
    while(str[j] != '\0') {
        vidptr[i] = str[j];
        vidptr[i+1] = 0x07;  // Attribute byte - light grey on black background
        ++j;
        i = i + 2;
    }

    return;
}
